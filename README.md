# Humps

This is a typescript translation of https://github.com/nficano/humps, which was itself inspired by https://github.com/domchristie/humps.

It seems like the nficano/humps is no longer maintained, and I had to make some minor corrections to the code when I translated it, following some pull requests that have been ignored by the original author [here](https://github.com/nficano/humps/pull/302/commits). All tests are now passing.
